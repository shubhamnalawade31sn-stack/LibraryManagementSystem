# Library Management System (Web API)

## Description
This is a simple Library Management System built using ASP.NET Core Web API and Entity Framework Core.  
It performs basic CRUD operations for managing books.

## Technology Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- Swagger

## Features
- Add a book
- Get all books
- Get book by ID
- Update book
- Delete book

## API Endpoints
- GET /api/books
- GET /api/books/{id}
- POST /api/books
- PUT /api/books/{id}
- DELETE /api/books/{id}

## How to Run
1. Open the solution in Visual Studio
2. Run the project
3. Use Swagger to test APIs
